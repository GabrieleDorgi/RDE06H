Software de Versionamento

Definição

Sistema que tem por objetivo realizar o controle de versões de um código ou documento.

Software de controle de versões

Cria a partir do arquivo um histórico de versões

Tipos de Software de controle de versões

  • Centralizado:possui um repositório central em um servidor com controle de versões. É dependente do servidor.
  • Distribuído:possui um repositório local com controle de versões. Atualizações no repositório central.Independente do servidor.

Git

Software livre de versionamento.Cada diretório de trabalho do git é um repoitório com um histórico completo e habilidade total de acompanhamento das revisões,não dependente de acesso a uma rede ou a um servidor central.

GitHub

Plataforma de hospedagem de código-fonte e arquivos com controle de versão.

Comandos Git

  • commit:torna permanente um conjunto de alterações de um repositório;
  • push:envia alterações para o servidor no repositório remoto;
  • fetch:verifica se o seu repositório local está em sincronia com o remoto;
  • pull:sincroniza o repositório local com o remoto atualizando as versões.
